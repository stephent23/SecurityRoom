Useful Tools

Virus Total

Service that allows the free lookup and predominently static analysis of malware samples and suspicious URLs, highlighting which vendors are already protecting against the threat.

Malwr

A service that offers full dynamic malware analysis and provides results very quickly. Also, has a great open source intelligence (OSINT) engine.

urlQuery

A service for testing and analysing URLs. It also provides the ability to be able to view screenshots of the webpages. Also, it provides an API to submit URLs.

About: The Site

Eventually, I hope that this will be a forum in which the latest Information and Cyber Security news can be discussed. I hope to be able to share some of the things that I have been working on and that the articles featured, will provide an insight into emerging threats, the challenges faced and encourage others to explore the exciting space of Information and Cyber Security. I hope that the articles will be appealing to all skill sets within the Information Security sector and also those that may have an interest in starting out in the industry. If you have any articles that you would like to contribute, topic suggestions that you would like to see discussed or any bugs on the website then please get in contact using the links on the navigation bar.

About: Stephen Tate

I currently works as an Information Security Analyst within the Finance sector, having previously worked within the security organisation of a large pharmaceutical company. I graduated from the University of Kent in 2015, where I studied Computing and first developed a passion for Information Security. I have a particular interest in emerging threats, advanced attacks and incident response.
